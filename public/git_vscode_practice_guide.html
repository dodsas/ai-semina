Git · VS Code · AI CLI 설치 가이드

실습에 필요한 Git 설치, Visual Studio Code 설치, 그리고 VS Code 터미널에서 AI CLI(Claude / Gemini)를 연동하는 방법까지 한 번에 정리한 자료입니다. Git 사용 방법(주요 명령어)은 별도 가이드를 참고하세요.

1

Git 설치

Git은 소스코드를 저장하고 변경 이력을 관리하기 위한 도구입니다. 실습에서는 Git을 사용해 예제 프로젝트를 다운로드하거나, 작성한 코드를 저장하고 서버에 업로드합니다.

Git for Windows 다운로드 Windows 환경에서 Git을 설치할 때 사용하는 공식 다운로드 페이지입니다.
다운로드

설치 순서

  1. Git for Windows 다운로드 페이지에 접속합니다.
  2. Windows용 설치 파일을 다운로드합니다.
  3. 다운로드한 설치 파일을 실행합니다.
  4. 설치 옵션은 대부분 기본값 그대로 진행합니다.
  5. 설치가 완료되면 Git Bash 또는 명령 프롬프트(cmd)를 실행합니다.
  6. 아래 명령어로 설치 여부를 확인합니다.
git --version

정상 설치된 경우 아래와 비슷하게 Git 버전이 출력됩니다.

git version 2.xx.x
2

Visual Studio Code 설치

Visual Studio Code는 코드를 작성하기 위한 개발 도구입니다. 실습에서는 VS Code를 사용해 프로젝트 파일을 열고 코드를 수정합니다.

Visual Studio Code 다운로드 Windows, macOS, Linux용 VS Code를 다운로드할 수 있습니다.
다운로드

설치 순서

  1. Visual Studio Code 다운로드 페이지에 접속합니다.
  2. Windows 버전을 다운로드합니다.
  3. 설치 파일을 실행합니다.
  4. 설치 옵션은 기본값으로 진행해도 됩니다.
  5. 설치가 완료되면 Visual Studio Code를 실행합니다.
3

AI 연동 공통 준비 — Node.js & 터미널

이제 VS Code 터미널에서 AI CLI(Claude / Gemini)를 설치합니다. 두 CLI 모두 Node.js(npm) 환경에서 설치하므로, 먼저 Node.js가 준비되어 있어야 합니다.

Node.js 다운로드 (LTS) Claude/Gemini CLI 설치에 필요한 npm 이 함께 설치됩니다. (Node.js 20 이상 권장)
다운로드

① VS Code에서 터미널 열기

Visual Studio Code 상단 메뉴에서 View(보기) > Terminal(터미널) 을 선택합니다. (단축키: Ctrl + ` — 백틱)

화면 하단에 터미널 창이 열리면, 앞으로의 명령어는 모두 이 터미널에 입력합니다.

② Node.js 설치 확인

터미널에 아래 명령어를 입력해 정상 설치 여부를 확인합니다.

node --version
npm --version

버전 숫자가 출력되면 준비 완료입니다. (예: v20.x.x)

버전이 안 보이거나 오류가 난다면

Node.js 설치 후 VS Code를 완전히 종료했다가 다시 실행한 뒤 터미널을 새로 열어주세요. (환경변수 반영)

Claude CLI 설치

Claude Code는 터미널에서 Claude 와 대화하며 코드를 작성·수정할 수 있는 Anthropic 공식 CLI 입니다. 아래 명령어는 VS Code 터미널(View > Terminal) 에 입력합니다.

① 설치
npm install -g @anthropic-ai/claude-code

npm 으로 Claude Code CLI 를 전역(-g) 설치합니다.

설치 중 에러가 난다면

아래 명령어를 먼저 실행한 뒤 다시 설치를 진행해 보세요.

npm config set strict-ssl false
설치 후 안내

설치가 끝나면 VS Code(또는 터미널)를 완전히 껐다가 다시 켜야 claude 명령이 인식될 수 있습니다.

② 설치 확인
claude --version

버전 숫자가 출력되면 설치가 완료된 것입니다.

③ 실행 & 로그인
claude

처음 실행하면 로그인 안내가 나타납니다. 화면 안내에 따라 Claude 계정으로 로그인(브라우저 자동 열림)하면 됩니다. 로그인 후 다시 claude 를 실행해 사용합니다.

참고

작업할 프로젝트 폴더에서 실행하는 것이 좋습니다. VS Code에서 프로젝트 폴더를 연 뒤 터미널에서 claude 를 실행하면 해당 폴더 기준으로 동작합니다.

Gemini CLI 설치

Gemini CLI는 터미널에서 Google Gemini 모델을 사용할 수 있는 공식 오픈소스 CLI 입니다. 아래 명령어는 VS Code 터미널(View > Terminal) 에 입력합니다.

① 설치
npm install -g @google/gemini-cli

npm 으로 Gemini CLI 를 전역(-g) 설치합니다.

설치 중 에러가 난다면

아래 명령어를 먼저 실행한 뒤 다시 설치를 진행해 보세요.

npm config set strict-ssl false
설치 후 안내

설치가 끝나면 VS Code(또는 터미널)를 완전히 껐다가 다시 켜야 gemini 명령이 인식될 수 있습니다.

② 설치 확인
gemini --version

버전 숫자가 출력되면 설치가 완료된 것입니다.

③ 실행 & 로그인
gemini

처음 실행하면 인증 방법을 묻습니다. Google 계정으로 로그인(Login with Google)을 선택하면 브라우저가 열리고, 로그인하면 바로 사용할 수 있습니다.

참고

설치 없이 한 번만 실행해보려면 npx https://github.com/google-gemini/gemini-cli 도 가능합니다. 작업할 프로젝트 폴더를 VS Code에서 연 뒤 터미널에서 gemini 를 실행하세요.
